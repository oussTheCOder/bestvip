## Avant de commencer

Avant d'activer IPTV sur votre Smart TV, votre fournisseur vous enverra généralement l'un de ces formats de connexion :

- **Xtream Codes API :** nom d'utilisateur, mot de passe et URL du serveur.
- **URL M3U :** un lien de playlist contenant vos chaînes IPTV.
- **Device ID, Device Key ou adresse MAC :** utilisé par des applications comme IBO Player et Ibo Player Pro.

Ouvrez la boutique d'applications de votre Smart TV et recherchez **IBO Player**. Installez l'application, puis ouvrez-la.

### Étape 2 : Trouver votre Device ID et Device Key

Lorsque vous ouvrez IBO Player, vous devriez voir un écran affichant :

Sur votre téléphone, tablette ou ordinateur, visitez le [site officiel d'IBO Player](https://iboplayer.com/).

Après cela, ajoutez votre playlist IPTV en utilisant l'une de ces méthodes :

- **URL M3U :** collez le lien de votre playlist.
- **Xtream Codes API :** saisissez l'URL du serveur, le nom d'utilisateur et le mot de passe.

![Ajouter une playlist sur le site Web IBO Player](/images/blog/07-ibo-player-step-3-add-playlist.webp)

### Étape 4 : Sauvegarder la playlist et redémarrer IBO Player

Après avoir ajouté votre playlist, sauvegardez les informations sur le site Web.

Revenez ensuite à votre téléviseur et redémarrez IBO Player. Vos chaînes devraient apparaître après l'actualisation de l'application.

Si les chaînes n'apparaissent pas immédiatement, fermez complètement l'application et rouvrez-la.

![Sauvegarder la playlist et redémarrer l'application IBO Player](/images/blog/08-ibo-player-step-4-save-restart.webp)

## Problèmes courants d'activation IPTV

### L'application n'est pas disponible sur ma Smart TV

Certaines applications ne sont pas disponibles dans tous les pays ou sur tous les modèles de téléviseurs. Si vous ne trouvez pas l'application sur votre LG TV, Samsung TV ou autre Smart TV, vous pouvez utiliser un appareil externe tel que :

- Boîtier Android TV
- Fire Stick
- Google TV
- Smartphone ou tablette Android

### Mes chaînes ne se chargent pas

Si vos chaînes ne se chargent pas, vérifiez les points suivants :

- Votre connexion Internet est stable.
- Votre nom d'utilisateur et mot de passe sont corrects.
- L'URL de votre serveur est correctement copiée.
- Votre URL M3U est complète.
- Votre abonnement ou essai est toujours actif.

Si vous utilisez Streambe, contactez le support ici : [Support IPTV Streambe](https://wa.me/19295664890?text=Hello%20Streambe%2C%20my%20IPTV%20channels%20are%20not%20loading).

### La playlist est sauvegardée mais rien n'apparaît

Si la playlist est sauvegardée mais que les chaînes n'apparaissent pas, redémarrez complètement l'application.

Vous pouvez également redémarrer votre téléviseur et votre routeur, puis rouvrir l'application.

Pour IBO Player et Ibo Player Pro, assurez-vous d'avoir utilisé le bon Device ID, la bonne Device Key ou la bonne adresse MAC.

### L'application demande une activation

IBO Player et Ibo Player Pro peuvent demander une activation de l'application après la période d'essai. Cette activation est distincte de votre abonnement IPTV.

Votre fournisseur IPTV vous donne la playlist ou les informations d'abonnement. L'activation de l'application est gérée par le fournisseur de l'application.
